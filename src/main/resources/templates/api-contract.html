API Contract - Jangkau

Base Url : https://koyeb-db-preview-app-jangkau-81ff3049.koyeb.app/api

POST /auth

Login

Request Body: Data pengguna baru yang akan ditambahkan.

{
  "username": "johndoe123",
  "password": "JohndoeEE23"
}

Responses:

200: Login Berhasil!

{
  "full_name": "John Doe",
  "email": "johndoe@example.com",
  "phone_number": "08223456766567"
}

400: Bad Request

GET /users/{id}

Get User By ID

Parameters: id (string) - ID user

Responses:

200: User ditemukan.

{
  "full_name": "John Doe",
  "email": "johndoe@example.com",
  "phone_number": "08223456766567"
}

404: User tidak ditemukan

GET /bank-accounts

Get Account Bank

Responses:

200: Daftar pengguna berhasil diambil.

[
  {
    "owner_name": "John Doe",
    "account_number": "8923445590",
    "type": "Tabunganku",
    "balance": 5450567
  }
]

GET /bank-accounts/{user_id}

Get Account Bank By User ID

Parameters: user_id (string) - ID user

Responses:

200: Bank Account ditemukan.

{
  "owner_name": "John Doe",
  "account_number": "8923445590",
  "type": "Tabunganku",
  "balance": 5450567
}

404: Bank Account tidak ditemukan

GET /bank-accounts/{account_number}

Get Account Bank By Account Number

Parameters: account_number (string) - Nomor rekening

Responses:

200: Bank Account ditemukan.

{
  "owner_name": "John Doe",
  "account_number": "8923445590",
  "type": "Tabunganku",
  "balance": 5450567
}

404: Bank Account tidak ditemukan

POST /transaction

Transfer

Request Body: Data transaksi ditambahkan

{
  "rekening_sumber": "8923445590",
  "rekening_tujuan": "8923445589",
  "amount": 540000,
  "transaction_date": "2024-07-12 08:26:37",
  "note": "Uang kos",
  "is_saved": false
}

Responses:

200: Transaksi Berhasil!

{
  "rekening_sumber": "8923445590",
  "rekening_tujuan": "8923445589",
  "amount": 540000,
  "transaction_date": "2024-07-12 08:26:37",
  "note": "Uang kos"
}

400: Bad Request

GET /transaction/{transaction_id}

Get Transaction By Transaction ID

Parameters: transaction_id (string) - ID transaksi

Responses:

200: Transaksi ditemukan.

{
  "rekening_sumber": "8923445590",
  "rekening_tujuan": "8923445589",
  "amount": 540000,
  "transaction_date": "2024-07-12 08:26:37",
  "note": "Uang kos"
